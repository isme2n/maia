# Portable-State Public Contract Matrix

This matrix is the Part 3 source of truth for Maia portable-state wording.

- `active`: must remain true in active public/help/test/code surfaces.
- `secondary`: supported public surface, but should not replace the primary `export` + safety-first `import` story.
- `legacy-history`: reserved for historical-only terms. No legacy Part 3 term is locked in this baseline.

## Notes

- The active Part 3 public path is `export` + safety-first `import`.
- `inspect` remains documented and supported, but follow-up docs/help/test work should keep it in a support role rather than making it look mandatory.
- The import safety contract is the combined behavior of preview output, destructive-import confirmation, and `--yes` as the explicit skip flag.
